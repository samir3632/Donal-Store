# 👕🛍️ Donal-Store

Tienda web de ropa para **hombre y mujer**, enfocada en ofrecer una experiencia de compra clara, moderna y dinámica.

---

## 📌 Descripción del proyecto

**Donal-Store** es una aplicación web para la gestión y visualización de productos de moda, diseñada para facilitar la navegación por categorías, la consulta de prendas y la interacción del usuario con la tienda.

El proyecto integra:

- Interfaz atractiva y responsive.
- Lógica back-end para gestión de contenidos/procesos.
- Base de datos para almacenamiento de información.
- Funcionalidades JavaScript para mejorar la experiencia del usuario.

---

## 🎯 Enfoque del sistema

La tienda está orientada a la comercialización de:

- **Ropa para hombre**
- **Ropa para mujer**

Incluye una organización visual y funcional para mostrar prendas por tipo, categoría o sección, mejorando la experiencia de búsqueda y compra.

## 📥 Instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/samir3632/Donal-Store.git
   ```

2. Entrar al proyecto:
   ```bash
   cd Donal-Store
   ```

3. Copiar la carpeta al entorno local (`htdocs` o `www`).

4. Configurar conexión a base de datos.

5. Crear/importar base de datos desde archivo `.sql` (si está incluido).

6. Ejecutar en navegador:
   ```txt
   http://localhost/Donal-Store
   ```
